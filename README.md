# fivem-lua-codes
Códigos desenvolvidos em lua utilizando a lib do cfx-fivem.


<br>
<br>

#

<br>

#
- Sistema para informar que o servidor será reiniciado. 


<br>
<br>

#QAWATER
#
- Sistema para remover itens do inventário ao entrar na água.
